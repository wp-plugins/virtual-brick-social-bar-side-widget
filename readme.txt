== Description ==

Easy To Use Social Bar with Your Favorite Social Share Buttons 

* Facebook Like + Share
* Twitter
* Google +1 
* Pinterest


== Installation ==

Activate the plugin, go to widgets page and add the widget to the desired location on the Sidebar.

* If Facebook Like button doesn't show check the Init Facebook checkbox 


== Frequently Asked Questions ==

= How to Configure my SocialBar ? =
Go to Admin -> Widgets, Select the Social Bar Widget and Expand 
Here you can select the title you want and enable or disable buttons

= Why is Facebook Button not Showing ? =
For facebook buttons to work you must have a facebook Init which your site might already have if you installed other facebook extensions
If not, mark the Facebook Init checkbox to solve this issue.
Note that this should only be done once, in case of multple widgets in page make sure you only activate the Init function on the first one.
